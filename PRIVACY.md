# Privacy Policy

This application does not collect, store, or share any 
personal information.

- No server-side data storage
- No cookies or session tracking
- Authentication handled entirely by Spotify
- Users must re-authenticate each session

# 개인정보 처리방침

## Spotify 데이터 사용 방법
- 인증은 전적으로 Spotify의 OAuth 시스템을 통해 처리됩니다
- 사용자의 선호도를 기반으로 플레이리스트를 생성하기 위해서만 Spotify 데이터에 접근합니다
- 사용자 정보는 서버에 저장되지 않습니다
- 데이터베이스 또는 서버 측 저장소를 사용하지 않습니다

## 세션 관리
- 매 세션마다 재인증이 필요합니다
- 쿠키나 영구 세션을 유지하지 않습니다
- 모든 데이터는 클라이언트 측에서만 처리됩니다

## 프로젝트 개요
**WeatherFlow Mix**는 스포티파이 사용자를 위한 개인 맞춤형 음악 발굴 도구입니다. 단순히 듣던 곡을 반복하는 것이 아니라, 사용자가 선호하는 아티스트의 다른 인기곡들을 탐색하여 새로운 취향을 찾아줍니다.

> **참고**: 현재 버전은 아티스트 기반 추천에 집중하고 있습니다. 날씨 기반 기능은 곧 추가될 예정입니다.

## 주요 기능
- **아티스트 마이닝**: 최근 즐겨 듣는 **상위 50곡**에서 아티스트를 추출합니다.
- **딥 디스커버리**: 해당 아티스트의 인기곡 중, 내가 이미 많이 들은 곡(Top 50)을 제외한 새로운 곡을 찾습니다.
- **자동 큐레이션**: 신선한 곡들 중 **30곡을 랜덤하게 선정**하여 플레이리스트를 구성합니다.
- **자동 동기화**: 클릭 한 번으로 내 스포티파이 계정의 `WeatherFlow Mix` 플레이리스트를 업데이트합니다.

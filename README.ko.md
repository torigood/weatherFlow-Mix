# 🎵 WeatherFlow Mix: 스포티파이 곡 추천 도구

**다른 언어로 읽기:** [🇺🇸 English Version](./README.md)

---

## 🚀 프로젝트 개요
**WeatherFlow Mix**는 사용자의 **상위 50곡**을 분석하여 선호 아티스트를 파악한 뒤, 해당 아티스트들의 곡 중 사용자가 평소에 자주 듣지 않았던 인기곡들을 찾아줍니다.

## ✨ 핵심 기능
- **스마트 필터링**: 현재 즐겨 듣는 상위 50곡을 결과에서 자동 제외.
- **아티스트 기반 로직**: 검증된 아티스트들의 'Top Tracks'를 교차 분석.
- **원클릭 업데이트**: 클릭 한 번으로 `WeatherFlow Mix` 플레이리스트 생성 및 동기화.
